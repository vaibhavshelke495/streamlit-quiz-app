# streamlit-quiz-app
The Streamlit Quiz App is an interactive web-based application developed using Python and Streamlit, designed to deliver a smooth and engaging quiz experience for users. The application presents multiple-choice questions in a clean and intuitive interface, allowing users to select answers, navigate through questions, and receive instant feedback in the form of a final score.

The project focuses on implementing core Python programming concepts such as conditional logic, loops, data structures (lists and dictionaries), and state management to track user responses and calculate scores accurately.
